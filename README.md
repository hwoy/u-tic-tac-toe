# u3t

u3t is micro Tic-Tac-Toe game (or called Unbeatable Tic-Tac-Toe Game).

## Features

1. Full open source.
2. Cross platform ability ,100% of ANSI C compatible (Syntax and Header files).
3. Not require special library.
4. Unbeatable AI.
5. Provides PvE, PvP, EvE.

Try it if you can!
